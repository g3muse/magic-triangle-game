# magic-triangle-game
Little game where user inputs 3 numbers, and the triangle works it's magic.
 
>> Preview:
>>        ( ? )
>>    [ x ]   [ y ]
>>( ? )   [ z ]   ( ? )

User is asked to input a number for each [  ] for x , y , and z.

Each ( ? ) adjacent to the [  ] must have a sum equal to the [  ] they are adjacent to.

>>Example:
>>        ( 5 )
>>    [ 7 ]   [ 5 ]
>>( 2 )   [ 2 ]   ( 0 )
>>
>>User entered: [7, 5, 2]
>>Answers:
>>Top:   5
>>Left:  2
>>Right: 0
